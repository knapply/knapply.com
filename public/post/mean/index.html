R

r_vector <- c(126, 793, 999, 525, 851, 798, 120, 714, 852, 151,
              190, 41, 612, 826, 507, 105, 14, 237, 669, 7)

Naïve Approach

Sum

Calculating the sum “from scratch”, using a loop:

r_sum <- function(x) {
  out <- 0
  for (i in x) {
    out <- out + i
  }
  out
}

r_sum(r_vector)
#> [1] 9137

The preferred, built-in solution:

sum(r_vector)
#> [1] 9137

Length

Finding the length “from scratch”, using a loop:

r_length <- function(x) {
  out <- 0
  for (i in x) {
    out <- out + 1
  }
  out
}

r_length(r_vector)
#> [1] 20

The preferred, built-in solution:

length(r_vector)
#> [1] 20

Put it all together

r_mean <- function(x) {
  sum(x) / length(x)
}

r_mean(r_vector)
#> [1] 456.85

Preferred Solution

mean(r_vector)
#> [1] 456.85

Rcpp

Naïve Approach

Length

Finding the length “from scratch”…

  • using a loop with NumericVector::iterator
// [[Rcpp::export]]
int cpp_length1(NumericVector x) {
  int out = 0;
  for (NumericVector::iterator it = x.begin(); it != x.end(); ++it) {
    out++;
  }
  return out;
}
cpp_length1(r_vector)
#> [1] 20
  • using std::distance() from <iterator>
// [[Rcpp::export]]
int cpp_length2(NumericVector x) {
  return std::distance(x.begin(), x.end());
}
cpp_length2(r_vector)
#> [1] 20
  • using std::vector::size() from <vector>
// [[Rcpp::export]]
int cpp_length3(NumericVector x) {
  return x.size();
}
cpp_length3(r_vector)
#> [1] 20

Note: If you’re using C++17 and beyond, consider using the more general std::begin(x), std::end(x), and std::size(x) instead of x.begin(), x.end(), and x.size().

Sum

Calculating the sum from scratch…

  • using a simple loop
// [[Rcpp::export]]
double  cpp_sum1(NumericVector x) {
  double out = 0;
  for (int i = 0; i < x.size(); i++) {
    out += x[i];
  }
  return out;
}
cpp_sum1(r_vector)
#> [1] 9137
  • using a loop with NumericVector::iterator
// [[Rcpp::export]]
double cpp_sum2(NumericVector x) {
  double out = 0;
  for (NumericVector::iterator it = x.begin(); it != x.end(); ++it) {
    out += *it; // `*` dereferences `it` so you can access the value to which it points
  }
  return out;
}
cpp_sum2(r_vector)
#> [1] 9137
  • using std::accumulate() from <numeric>
// [[Rcpp::export]]
double cpp_sum3(NumericVector x) {
  return std::accumulate(x.begin(), x.end(), 0.0);
}
cpp_sum3(r_vector)
#> [1] 9137

The preferred Rcpp sugar solution:

// [[Rcpp::export]]
double cpp_sum4(NumericVector x) {
  return sum(x);
}
cpp_sum4(r_vector)
#> [1] 9137

Put it all together

// [[Rcpp::export]]
double cpp_mean_naive(NumericVector x) {
  return sum(x) / x.size();
}
cpp_mean_naive(r_vector)
#> [1] 456.85

Preferred Solution

Using Rcpp sugar:

// [[Rcpp::export]]
double cpp_mean(NumericVector x) {
  return mean(x);
}
cpp_mean(r_vector)
#> [1] 456.85

Python

py_list = [126, 793, 999, 525, 851, 798, 120, 714, 852, 151,
           190, 41, 612, 826, 507, 105, 14, 237, 669, 7]

Naïve Approach

Sum

Calculating the sum “from scratch”, using a simple loop:

def py_sum(x):
  out = 0
  for i in x:
    out += i
  return out

res = py_sum(py_list)
print(res)
#> 9137

The preferred, built-in solution:

res = sum(py_list)
print(res)
#> 9137

Length

Finding the length “from scratch”, using a simple loop:

def py_length(x):
  out = 0
  for i in x:
    out += 1
  return out

res = py_length(py_list)
print(res)
#> 20

The preferred, built-in solution:

res = len(py_list)
print(res)
#> 20

Put it all together

def py_mean(x):
  return sum(x) / len(x)
  
res = py_mean(py_list)
print(res)
#> 456.85

Preferred Solution

Using Python’s built-in statistics module:

import statistics as stat

res = stat.mean(py_list)
print(res)
#> 456.85

Using numpy:

import numpy as np

numpy_array = np.array(py_list)
print(numpy_array)
#> [126 793 999 525 851 798 120 714 852 151 190  41 612 826 507 105  14 237 669   7]
res = np.mean(numpy_array)
print(res)
#> 456.85
